Here, I am introducing some of my work integrated based on service, HTTP, and pipe.
A data table below contains the user's details, but the details are not static here.
Yes! They are fetched from a JSON file dynamically. To fetch the data from a JSON file I have used the service.
The data is being sorted by Date Of Birth. The sort has been implemented using the pipe (which can be used anywhere throughout the application).
The last column 'Age' is not available in a JSON file. To display it in the table, I have implemented a pipe, which takes a dob from the user-list and calculate the age on the current date.
